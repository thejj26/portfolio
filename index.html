About Me

I'm Jakov Jozić, a 3rd grade student at the School of Electrical Engineering in Split, Croatia. I've had a passion towards programming since I was young. It started as some QBasic and Pyhton programming which was a part of my elementary school curriculum. I took some courses but later gave up on programming.

My love towards programming was reawakened when i took part in the EDIT Code School course in 2020. It was what got me into web development. My project won2nd place that year but that wasn't enough so I went further. Next year I took on the advanced course and my project won 1st place. After that I knew for sure that web development was the way for me.
